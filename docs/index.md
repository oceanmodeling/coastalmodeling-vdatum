# Welcome to coastalmodeling-vdatum

`coastalmodeling-vdatum` is a Python package designed to support coastal modelers with vertical datum conversions.

This documentation provides a guide to using the package, including installation instructions, a quick start guide, and an API reference.

## Getting Started

To get started, please see the [Installation](./installation.md) and [Quick Start](./quick-start.md) guides.
